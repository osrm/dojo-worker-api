## Run Locally

Currently the repo is structured to house multiple microservices, where each of the microservices' `main` functions are in `/cmd/service_name/main.go`

### To run auth service
```bash
go run /cmd/auth/main.go
```

### To run task service
```bash
go run /cmd/task/main.go
```
